# THIRTY-ONE

A roguelike card game inspired by the classic card game 31.

## Features

- 9 Decks, 8 Ascension levels, 32 Modifiers
- 12 Relics, 8 Spirits, 5 Card Marks
- 18 Bosses, 7 Challenges
- Daily & Endless modes
- 10 Achievements

## New in this version

- **Card Preview in Shop**: Hover/tap upgrade/mark items to see your deck cards
- **Spirit Master Achievement**: Get 3 loyal spirits in one run to unlock The Void relic
- **Spirit indicators**: Shows loyalty progress in gameplay UI

## Deploy to GitHub Pages

Push to repo → Settings → Pages → Deploy from main

## Local Dev

```bash
npx serve
```
